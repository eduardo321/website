Audacity allows you to add effects to any sound that you make. Some examples of these sound effects are changing the pitch, which makes the sound higher or lower. Another effect is called 'reverse' which just makes the end the beginning and the beginning the end. This can be quite amusing when playing around with the words. You get many more effects such as changing the speed which will decrease the amount of total playing time. To find all of the effects that Audacity has to offer you just simply go on effects and they will be there.

Audacity allows you to select any given bit of the song, so that you can play a particular bit of the song back and you can add effects to a particular bit, such as at the end you where you could use the fade out effect, which just makes it drown out and gradually quieter.

Audacity can be used to import and export other file types such as WAV, MP3 and others. To do this all you have to do is file, import, audio and then select the file you want and it will be in Audacity, where you will be able to change and edit it how you like. All you do to export is save it as the file type you want such as MP3 and then you can open it as an MP3 file when you want.

The effect that I will be using is the change speed effect.

When you compare the difference between the first track and the second track, you can tell that the second track is shorter and plays it faster. This is because it increased the speed and by doing this it also increased the frequency. Speed and frequency are linked together.

There is a big difference between the two tracks, due to the fact that they are a couple of effects that are being used, such as fade in, fade out, change speed and more.
