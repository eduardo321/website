report

I made the logo myself using Photoshop which I found quite changeling to do a first, but will spending more time on it I did it in the end.

I made the page layout using, CSS and selected the colour scheme, which was mainly shades of blue due to the fact the restaurant is a fish one. I am pleased with the way that the web pages look. The only decision that I had to make was the different blues I was going to use. I also used CSS to design the navigation bar and made it with rounded corners, which has red around because the red to the blue provides good contrast, which makes it clear to see, similar to the words, which are black and the background is blue. I added a hover feature for the navigation bar, which means when you hover over part of the bar it will change colour, which tells you the page that you will go to if you click it.

To design the menu I used a xml file and styled it using css and used javascript code to make it display on the HTML page that I wanted. When I made changes to the CSS code, the XML was not changing in HTML. It took be a while to figure out why it was not working, but when the problem was solved it was quite easy

I used Dreamweaver validation for validating the different forms that I used, which is in the form of JavaScript. My validations were not working correctly, to begin with, but once I changed to onclick it worked.

To create the buttons for the next and previous for the slide show I used HTML,t o design them I used CSS, and to make the buttons work I used Jquery and Jquery library’s for it to work. I found this quite simply to do and did not have any major difficulties doing this

To make the mobile app I found quite tricky as it took me a while to figure out the default, such as where to put things, so I could design it how I wanted to which is similar to my website pages. I created the mobile app using phone gap. I also created a mobile app preview via a link so you can what it will look like when you download it on your mobile phone. I made a icon that when you scan it with your phone

The website could be improved is the slideshow for the mobile app is not working as when you press the previous and back buttons they do not work. I could have added captions for the slide show with a description of what each picture is represents. I also could have improved the website by using meta-tags for all the web pages. One other way that I could have improved the website, is for the slide show I could have added more buttons such as buttons which take the slide show to the first picture and another one that shows the last picture.
